##############################################
#              USER GUIDE                    #
##############################################

##############################################
# 1. Accessing the Platform                  #
##############################################
# 1. Open your browser and navigate to the home landing page:
#    http://www.clothingtradingplatform.com
#    (Note: If your browser automatically adds an 's' to http, the page will not load. You must revert it back to http.)
# 2. If you already have an account, click the Login button to proceed.
# 3. If you do not have an account, click the Register button to create a new account.

##############################################
# 2. Registering an Account                  #
##############################################
# 1. On the Register page:
#    - Enter a valid username.
#    - Provide your email address.
#    - Choose a secure password (minimum 8 characters).
# 2. Click the Register button to create your account.
# 3. Once registered, you will be redirected to the Login page.

##############################################
# 3. Logging In                              #
##############################################
# 1. On the Login page:
#    - Enter your username and password.
# 2. Click the Login button to access your account.
# 3. Upon successful login, you will be directed to the Dashboard.

##############################################
# 4. Dashboard Overview                      #
##############################################
# The Dashboard provides an overview of your account and activities:
# - Account Information: Displays your username and account details.
# - Recent Trades: Lists the most recent trades you have participated in.
# - Recent Clothing Items: Displays a summary of your recently added clothing items.
# - Admin Features (if applicable):
#   - Admin users can generate and download a Trade Report directly from the dashboard.

##############################################
# 5. Managing Clothing Items                 #
##############################################
# 1. Navigate to the My Clothing Items section using the navigation bar.
# 2. Adding New Items:
#    - Fill in the form with details such as:
#      * Title: The name of the item.
#      * Description: A brief description of the item.
#      * Size, Brand, Condition, and Price.
#    - Click Add Item to save the item.
# 3. Editing Existing Items:
#    - Click Edit on an existing item to make changes.
#    - Update the details as needed and click Save.

##############################################
# 6. Searching for Clothing Items            #
##############################################
# 1. Use the Search Bar on the top of the page to quickly search for items by name.
# 2. Alternatively, navigate to the Search Page:
#    - Use the filters to refine your search:
#      * Title, Brand, Size, Condition, Minimum Price, Maximum Price.
#    - Click Apply Filters to display matching items.
# 3. Reset Filters: Click the Reset Filters button to clear all applied filters.

##############################################
# 7. Viewing Item Details and Requesting Trades
##############################################
# 1. From the search results or any item list, click View Details on an item you’re interested in.
# 2. On the item details page:
#    - View more information about the item (e.g., description, condition, price).
#    - Click Request Trade to initiate a trade with the item’s owner.

##############################################
# 8. Managing Trades                         #
##############################################
# 1. Navigate to the My Trades section from the navigation bar.
# 2. View your trades categorized as:
#    - Initiated Trades: Trades you have requested on other users' items.
#    - Received Trades: Trades requested by other users on your items.
# 3. For Received Trades:
#    - You can choose to Accept or Decline a trade.
#    - Once a trade is accepted, the requesting party’s email will be visible for further communication.

##############################################
# 9. Trade Completion and Cancellation       #
##############################################
# 1. Completing Trades:
#    - Once both parties are satisfied with the arrangement (e.g., payment and shipping), either party can mark the trade as Complete.
#    - Once completed:
#      * The item will no longer be available for trade on the platform.
#      * The original owner will still see the item as "Unavailable" in their clothing items list.
# 2. Cancelling Trades:
#    - Either party can cancel a trade at any time if it has not yet been marked as complete.

##############################################
# Additional Notes                           #
##############################################
# - Communication:
#   - After accepting a trade, you will need to contact the other party via email to finalize payment and shipping details (internal messaging will be added in a future version).
# - Trade History:
#   - Your initiated and received trades remain visible under the My Trades section for tracking.
# - Images:
#   - The current version does not support uploading pictures. Later iterations will allow picture uploads with filters to block inappropriate content.
